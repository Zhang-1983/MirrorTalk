# MirrorTalk 项目更新记录

## 项目概览

MirrorTalk 是一个基于 React + TypeScript + Node.js + Express + Prisma + SQLite 的企业沟通培训系统，旨在帮助管理者提升沟通技巧。

## 系统功能

### 1. 用户功能
- 注册和登录
- 个人资料管理
- 训练历史查看

### 2. 管理员功能

- 系统监控

## 技术栈

- **前端**：React + TypeScript + Vite
- **后端**：Node.js + Express + Prisma + SQLite
- **认证**：JWT 令牌
- **API**：RESTful API
- **实时通信**：Socket.io
- **AI**：DeepSeek API
- **密码哈希**：bcryptjs
